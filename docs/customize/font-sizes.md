# Font sizes

The font sizes of the different elements of the timeline card can be customized via the `fontSizes` prop.

```jsx
  <Chrono
    items={data}
    mode="HORIZONTAL"
    fontSizes={{
      cardSubtitle: '0.85rem',
      cardText: '0.8rem',
      cardTitle: '1rem',
      title: '1rem',
    }}
  />
```
